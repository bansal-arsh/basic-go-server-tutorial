# Basic Go Server

This is a basic HTTP API server written entirely in Go. It also simulates a database which the server uses to store user data. The server exposes endpoints to add and retrieve users from the server. The server also exposes many endpoints which take in input using various methods, namely:
- URL parameters
- Custom headers
- Variable URL paths
- Request payload (for JSON queries and POST requests)

Huge thanks to **Mongoose Studios** for their [YouTube tutorial](https://www.youtube.com/watch?v=lyh9G3-Q7PQ&list=PLlv5lnjOHQo7m2w-KjtTZq10ZNVfNHHdP&index=1) on setting up a Go HTTP Server. This server was made following their tutorial.
